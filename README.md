# ChatGPT UX Improvements

This repository contains a collection of Chrome extensions designed to improve the user experience on ChatGPT.

## Extensions

### 1. ChatGPT Context Counter
**Location:** `chatgpt_context_counter_extension/`

Displays an estimated token count for the current ChatGPT conversation.

### 2. ChatGPT Prompt Navigator
**Location:** `chatgpt_prompt_navigator_extension/`

Allows you to quickly navigate between user prompts in a conversation.
*   **Jump to Previous Prompt:** `Alt+E`
*   **Jump to Next Prompt:** `Alt+D`

### 3. ChatGPT Styling Extension
**Location:** `chatgpt_styling_extension/`

Adds custom styling to the ChatGPT interface to improve readability and eye tracking. It's also recommended to change your theme color, which changes background of user prompts.
*   **Model Responses:** Styled with a translucent "soft blue rectangle".

## Installation

To install any of these extensions:

1.  Clone or download this repository.
2.  Open Chrome and navigate to `chrome://extensions/`.
3.  Enable **Developer mode** in the top right corner.
4.  Click **Load unpacked**.
5.  Select the directory of the specific extension you want to install (e.g., `chatgpt_context_counter_extension`).
